# Data Table UX for Non-Technical Users

## Research Summary

Based on analysis of leading no-code tools:
- [Airtable Linked Records](https://www.airtable.com/guides/build/connect-data-with-linked-records) - Visual linking between tables
- [Notion Relations & Rollups](https://www.notion.com/help/relations-and-rollups) - Database connections with aggregations
- [Visual Query Builders](https://www.activequerybuilder.com/product_ui.html) - Drag-drop query construction
- [Microsoft Fabric Visual Editor](https://learn.microsoft.com/en-us/fabric/data-warehouse/visual-query-editor) - No-code data querying

**Key Insight**: Non-technical users should NEVER write queries. They interact with visual interfaces that generate VDQL behind the scenes.
