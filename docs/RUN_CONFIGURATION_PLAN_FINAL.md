## API Endpoints (New/Updated)

### GitHub Integration

```
POST   /api/github/connect           # OAuth callback
POST   /api/github/validate-token    # Validate PAT
GET    /api/github/repos             # List user's repos
GET    /api/github/repos/:id/branches

POST   /api/github/workflows/generate   # Generate workflow file content
POST   /api/github/workflows/create-pr  # Create PR with workflow file

POST   /api/github/runs/trigger      # Trigger workflow_dispatch
GET    /api/github/runs/:id/status   # Poll run status
GET    /api/github/runs/:id/jobs     # Get job details
GET    /api/github/runs/:id/artifacts # List artifacts
GET    /api/github/runs/:id/artifacts/:name/download # Download artifact
```

### Execution (Updated)

```
POST   /api/executions              # Create execution (any target)
GET    /api/executions/:id          # Get execution with results
GET    /api/executions/:id/stream   # SSE for live updates
DELETE /api/executions/:id          # Cancel execution
```
